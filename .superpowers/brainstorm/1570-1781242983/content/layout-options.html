Dual-Column Layout Options

Left: Chat flow (messages, input). Right: Reasoning panel (thinking steps, charts, evidence).

A

60/40 Split

Chat takes 60%, reasoning panel 40%. More room for conversation, less for steps.

B

50/50 Split

Equal space. Balanced for both reading answers and observing the AI's thinking process.

C

70/30 Split

Chat 70%, reasoning 30%. Chat-focused with a compact side panel for steps.

Reasoning Panel Mode

How should the right-side reasoning panel behave?

A

Always Visible

Panel is fixed on the right. Shows current thinking step in real-time as the AI works.

B

Auto-Expand / Collapse

Panel is hidden until AI starts thinking, auto-expands. Collapses after answer is complete.

C

Togglable

User can manually toggle panel visibility. Panel retains last state across questions.
