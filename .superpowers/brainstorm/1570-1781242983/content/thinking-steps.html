Thinking Steps Visualization

How should the ReAct thought process look in the right panel?

A

Timeline Style

Vertical timeline with icons. Each step shows: dot + tool name + brief result. Running step pulses, completed steps get green checkmark. Clean and scannable.

🔵 Running: query_exam_performance — "通过率=61%..."
✅ search_course_or_exam — "反洗钱培训 → EXM001"
✅ query_completion_rate — "完成率=85%, 320人"
B

Accordion Cards

Each step is a collapsible card. Header shows tool name + status. Expand to see full thought, params, and result. Good for detailed debugging.

🔵 Step 2: query_exam_performance
scope_type=all, time_start=2026-01-01...
✅ Step 1: search_course_or_exam
C

Minimalist Dot Chain

Dots connected by lines. Hover to see tooltip with details. Minimal visual noise, maximum space for answer and charts.
